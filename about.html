About my site

The website design of MyPassion.com is an amazing journey for me to learn how to structure the website content. In the first stage, with respect to the sitemap, I tried to display a menu layout with enrich information. I used an image map to visualise the links in the second stage. By doing deeply investigation, I found that most functional websites contain the pages of home, about us, news, services, feedback and contact me. Thus, in stage 2, I learnt how to implement a dropdown menu with a list tag and how to line up a sitemap by using CSS, since the photo gallery, video and storyboard can be categorized in “My Story” menu.

For MyPassion.com, in my Home page I introduced myself and stated my
goals with reference to the website design. After the first two stages, I
revised my introduction regarding my empirical works that is more related
to the website content. I have designed a striking logo and a theme template
with my favourite colour. With the help from Freebie footer and font awesome
CSS file, a pretty and tidy footer is fitted in my website to present the contact
information and social media. In the stage 3, I started over to implement the social media buttons, but this time all the buttons can work. I also learnt how to float each division tag and make it display in correct position which is a tricky part in CSS.

Within the News menu, the RSS Feed Informer widget can assist with a neatly arranged feeds. However, it is slightly difficult to find the links of data source which should end up with xml.

Here comes My Story! The photo gallery was built up based on a photo wrapper. By designing two buttons in each side of the frame, I tried to use a cursor attribute in CSS. Meanwhile, it is essential to write a JavaScript to show to the images and auto play them in every 3 seconds. I also have done the labelling of the images by creating a caption division in the third stage. I enjoyed the way that I learnt all these methods by myself. I used the Comic Life 3 to create my storyboard. It is very interesting when I was mapping all the images together and building up my storyline, I recalled all the wonderful memories between me and my besties. Same as creating an inline video by using Movavi Video Editor, it was a funny moment to add any transitions to the video.

Designing a feedback form is a best way to learn how to create a table by using HTML, since it is a bit confusing to show how many rows and columns in it. In the Contact me page, I learnt how to show Google Map by implementing the API through an iframe tag. Most interesting thing is to create an image map by locating the area’s coordinate and redirecting the link to the corresponding menu.
